Evidencer MobileFirst Server is the back end of Evidencer Pro that controls all the client activities providing a safe and secure data management capability. Any third party tracking of data can be avoided since the data is completely secure in adapters which act as a mediator between mobile applications and enterprise system. Adapters provide an easy and secure access to enterprise system



  • It restricts the sharing of device used for accessing the Evidencer application and ensures whether the data being displayed is suitable for public share.
  • Provides appropriate use of firewalls and authentication feature for accessing and performing administration tasks which ensures the court data is safe within the authority.
  • MobileFirst Server is a scalable gateway between Evidencer and users along with other court management services providing strong security features.
  • The whole server arrangement is available in On-premise Applications as well as cloud platform which make way for enterprise mobility.
  • Native connectors and template integration processes (TIP’s) to connect mobile apps to backend and cloud systems, reducing project costs up to 80%.
  • Bidirectional connectivity and business logic to increase data quality and streamline business processes.
  • Centralized monitoring of court procedures.
  • Simple and flexible, user-friendly, wizard-based, “configuration, not coding” architecture provides best-practices and repeatable mobile integration.

Security and Privacy Features of Evidencer MobileFirst Server

  • Encrypted Offline Cache: Protect sensitive information from Malware attacks and device theft which is a major factor regarding judicial data.
  • Direct Update: Ensure timely propagation of critical security updates to the entire install base which enables updated time-to-time security feature.
  • Authentication Framework: Ability to lower overall cost and complexity of integration with authentication infrastructure.
  • Server-side safeguards: Prevent SQL Injection and protect against XSRF, a type of web attack where an attacker can perform actions on behalf of an authenticated user without user's knowledge.
  • Enterprise SSO Integration: Leverage existing enterprise authentication facilities and user credentials.
  • VPN Alternative: Enable secure delivery and operation of mobile apps for employee-owned devices or device types not allowed on the corporate network, as well as enable secure delivery when installation of VPN client on mobile devices is not possible or complicated to manage.

Capturing data and recording events looks like a menial task for any recording solution. But, playback of outputs from such solutions meant only for recording is different from what Evidencer offers. Evidencer is much more than your average recorder. The EviRecorder component in Evidencer is responsible for capturing audio and video data along with log notes that are captured throughout the recording.



  • EviRecorder does much more than just capturing data through 4, 8 or 16 independent audio and video channels.
  • It constantly monitors if the audio levels captured are of optimum quality.
  • EviRecorder also syncs audio, video and log notes through unified date and timestamps to ensure that when the recorded session is viewed, user can reach any event much faster.
  • Every recorded data session is automatically uploaded to the central server.
  • Since storage is of importance when such volumes of data capturing is involved, EviRecorder constantly monitors and performs hard disk checks to ensure that you never run out of memory.
  • EviRecorder provides a reliable and secure capability to capture data sessions that are of optimum quality.
