When sensitive processes such as courtroom proceedings, police interrogations and small court sessions are to be digitized and data needs to be captured, security and encrypting of data is of utmost importance. Evidencer provides host of features other than recording data. It also gears security by controlling the access to available features such as report generation, broadcasting and live transmissions through EviPortal. EviPortal comes with an intuitive UI and makes it easy even for users with limited technical knowledge to perform these tasks.



Portal Server

EviPortal provides authorized access to manage tasks in Evidencer software. Certain tasks and modules such as live streaming, broadcasting or generating reports can be accessed only by a set of authorized users. EviPortal lets you create separate groups for each of these tasks and provide them with certain privileges. All tasks related to granting access and authorization can be controlled by the admin through EviPortal.



Admin Module

Administrators are authorized to identify and control the state of users logged into the eviPortal. Administrator can give each user a set of roles that grant access to the different functions of eviPortal. Users’ privileges for their roles, access to specific functionality such as recording feature are set by the administrator. Case status, case type, locations, etc. can also be added or managed.



User Module

Users can see various analytic reports on the dashboard of eviPortal providing a visual report of data related to various cases.

  • Statistical Analytic Report: User can enter the case type like civil, criminal, family court case, probate case etc. Based on the case type enter a graph gets displayed based on number of cases vs. year.
  • Satisfactory Analytic Report: Report based on the rating given to the judge for his judgement.
  • Most Popular case: List with most popular case for that year.
  • Cases with maximum hearings: Case number vs. number of hearings for that case.


Video on Demand

EviPortal comes with provision to search the previously recorded case session and play back when needed. All session related details such as Lognotes, private notes other related files which are synched enables the user to readily access any particular part of case file as convenient. User authentication is provided to ensure that only authorized users can view the sensitive information like private notes. EviPlayer was designed to enhance the viewing capabilities of users and providing them with option to search for events in recorded session with ease.



Live Broadcasting

Live streaming is the present and future of the Web, and is currently enjoying a huge demand from online consumers. EviPortal's live streaming allows the sending and receiving of media across the World Wide Web in real-time. Thus, the user can watch a streaming video of court happenings as simple from remote location through EviPortal.
